# Hướng dẫn submit bài tập

- Checkout sang nhánh mới khi bắt đầu làm bài

```shell
git checkout -b ten_bai
```

- Sau khi hoàn thành commit và push lên git

```shell
git add .
git commit -m "đề bài"
git push origin tên_nhánh_đã_tạo
```
